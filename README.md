# Civara Landing Page

Standalone landing page for Civara with an early-access form wired for Supabase.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`

## Early Access Setup

1. Copy `.env.example` to `.env`.
2. Add your Supabase project URL and anon key.
3. Run the SQL in `SUPABASE_SETUP.md` to create the waitlist table and insert policy.
