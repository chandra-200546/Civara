# Supabase Early Access Setup

Create a table named `early_access_signups` in Supabase and run:

```sql
create table if not exists public.early_access_signups (
  id bigint generated by default as identity primary key,
  full_name text not null,
  email text not null,
  phone text not null,
  preparation_stage text,
  created_at timestamptz not null default now()
);

alter table public.early_access_signups enable row level security;

create policy "Allow anonymous inserts for early access"
on public.early_access_signups
for insert
to anon
with check (true);
```

Then add the values from `.env.example` into your local `.env` or deployment environment.
